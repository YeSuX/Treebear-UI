## 开关 Switch

### props

| 属性            | 说明                                 | 类型   | 默认值    |
| --------------- | ------------------------------------ | ------ | --------- |
| value / v-model | 默认值                               | bool   | false     |
| size            | 尺寸,可选值`large`,`normal`,`small` | string | 'normal' |
| disabled        | 是否禁用                             | bool   | false     |
| checkedText     | 打开时展示的内容, size为small时无效  | string | ''        |
| loading         | 设置加载状态                         | bool   | false     |

### events

| 事件名称 | 说明                            | 回调参数   |
| -------- | ------------------------------- | ---------- |
| change   | switch 状态发生变化时的回调函数 | 新状态的值 |
