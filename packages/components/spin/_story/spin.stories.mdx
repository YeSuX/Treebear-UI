## 加载器 Spins

| 属性      | 说明                                          | 类型       | 默认值 |
| --------- | --------------------------------------------- | ---------- | ------ |
| size      | 组件大小，可选值为 `small`, `middle`, `large` | string     | middle |
| tip       | 当 spin 作为包裹元素时，可以自定义描述文字    | string     | ''     |
| indicator | 自定义加载指示符                              | string     | spin   |
| delay     | 延迟显示加载效果的时间                        | number(ms) | 0      |
